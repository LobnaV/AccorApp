Conditions / rules to be respected:
  • Only the General Manager shall access the application and perform changes within users of his hotel (no Head of Department).
  • If some changes are temporary (holiday, maternity leave, sick leave), we encourage you use the delegation rules in place in Tradeshift instead of performing permanent changes.
  • If you need to change the name of one of your user (and keep the same email address), please edit directly the user in the corresponding field and do not delete the user to recreate one.
